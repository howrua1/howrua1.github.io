How to Install Exodus on Kodi 17.6

The first step to install Exodus on Kodi 17.6 is to enable unknown sources. To do this, go to the Settings icon on the Kodi homepage, click System Settings, and then click Add-ons. Next, click the toggle for Unknown sources and click Yes when prompted. Now, return to the Kodi homepage and click the Settings icon again.

From there, click on File Manager and select Add source. Enter http://lazykodi.com/ in the top box and “lazy” in the bottom box. Click OK and go back to the Kodi homepage. Select Add-ons and click the package installer icon in the top left corner. Click Install from zip file and select the lazy directory. Next, navigate to the ZIPS directory and click on “repository.kodil-1.3.zip”.

Now, click Install from repository, select Kodil Repository, and then click Video add-ons. Scroll down and click on Exodus, then click Install.
