Answer to the Target of Terraria How to Install Mods

How to Install tModLoader for Linux Users

If you are a Linux user, the process of installing tModLoader is a bit different. Follow the instructions below to install tModLoader on Linux:

  1. Download the Linux version of tModLoader from the official website.
  2. Extract the downloaded file to any directory you prefer.
  3. Open a terminal and navigate to the directory containing the extracted files.
  4. Run the following command: “chmod +x TerrariaServer*”
  5. Finally, run the tModLoader server by entering “./TerrariaServer -config serverconfig.txt”.
