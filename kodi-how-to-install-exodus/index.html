What is Exodus on Kodi and How to Install it?

Exodus is a Kodi add-on that allows users to stream movies and TV shows. It is known for its vast library of content and user-friendly interface. To install the Exodus add-on on Kodi, you need to follow these simple steps:

Step 1: Add the Kodi Bae Repository to Kodi

The first step is to add the Kodi Bae Repository to Kodi. To do this, go to the Kodi home screen, click on the Settings icon, then click on File Manager, and finally, click on Add Source. Here, you need to enter the URL for the Kodi Bae Repository: http://lazykodi.com/. Then, name the repository something easy-to-remember like “KodiBae” and click OK.

Step 2: Install the Kodi Bae Repository

After adding the Kodi Bae Repository to Kodi, you need to install it. For this, return to the Kodi home screen, click on Add-ons, then click on the Package Installer icon in the top-left corner, followed by Install from Zip File. Here, select KodiBae, and then the repository zip file called “repository.kodibae-x.x.zip”. Wait for the Repository to install, and you’ll receive a notification once it’s complete.

Step 3: Install Exodus on Kodi from Kodi Bae Repository

Once you have installed the Kodi Bae Repository, you can now install Exodus on Kodi. To do this, click on Install from Repository, then select Kodi Bae Repository , click on Video Add-ons and in the list find Exodus. Finally click on “Install”.
