How to Install Ares Wizard on Kodi 17.4

Installing Ares Wizard on Kodi 17.4 is actually quite simple once you know the steps. First, you’ll need to add the Ares Wizard repository to Kodi. Then, you can download and install Ares Wizard from the repository. Here is a step-by-step guide to help you install Ares Wizard:

  1. Open Kodi and navigate to Settings > File Manager > Add Source.
  2. Enter the URL http://ares-repo.eu/ and name it “Ares Wizard”.
  3. Go back to the Kodi home screen and select Add-ons > Package Installer (the box icon in the top left corner).
  4. Select Install from zip file and click on “Ares Wizard”.
  5. Navigate to the repository.aresproject folder, select the version of Kodi you’re using, and click on repository.aresproject-x.x.x.zip to install the repository.
  6. Go back to the Package Installer menu and select Install from repository > Ares Project > Program Add-ons > Ares Wizard > Install.
  7. Wait for the add-on to download and install. You’ll see a notification once it’s done.
  8. Restart Kodi to start using Ares Wizard.
