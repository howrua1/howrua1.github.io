How to Install Fusion

Installing Fusion on Kodi is relatively easy and straightforward, provided you follow the necessary steps. Here’s a step-by-step guide on how to install Fusion on Kodi to get you started.

Step 1: Launch Kodi and Click on the Gear Icon

Launch Kodi and click on the gear icon located in the upper-left corner of the Kodi home screen to navigate to the system page.

Step 2: Click on the File Manager

Once you’re on the System page, find the file manager and click on it to continue.

Step 3: Click on the Add Source

After clicking on the file manager, choose Add Source and then click on None to enter the URL of the source.

Step 4: Enter the Fusion URL

In the box labeled None, enter the following URL: http://fusion.tvaddons.ag and then click Ok.

Step 5: Name the Source

In the next box, give the media source a name. For example, you can name it Fusion or any other name that you can easily remember. Once you’re done, click Ok.

Step 6: Install the Add-On Repository

Once you’ve added the Fusion source, return to the Kodi home screen and navigate to the add-ons section. Select the open-box icon on the top left corner of the screen and then click on Install from zip file. Select the Fusion file you’ve just added, followed by the Kodi-repos folder, and select the English folder. Choose the fusion-repo-x.x.x.zip file and install it. Wait for the “Add-on Enabled” notification to appear before continuing.

Step 7: Install the Add-On from the Repository

After installing the add-on repository, go back to Install from repository and choose the Fusion Repo. Select the category of the add-on you want to install, such as video, music, or program, and install it. Once the installation process is complete, return to the add-ons section to find your newly installed add-on.

Question and Answer

Q: Is it legal to install Fusion on Kodi?

A: Yes, it is entirely legal to install Fusion on Kodi. However, it must be noted that the legality of third-party Kodi add-ons may vary depending on regional laws and regulations.

Q: Is Fusion and Indigo the same thing?

A: No, they are different things. Fusion is an open-source repository that allows you to download or install many third-party add-ons. Indigo is an add-on installer maintenance tool designed and maintained by Fusion repository.

Q: Do I need to install VPN before installing Fusion on Kodi?
