How to Install Pulse on Kodi

First, you’ll need to download the Pulse CCM repository, which can be found by doing a quick Google search. Once you have it downloaded, open Kodi and click on the gear icon in the top-left corner to access the settings menu. From there, click on “File manager” and then “Add source”. In the popup window, click on “None” and then type in the URL where you downloaded the Pulse CCM repository. Give it a name (like “Pulse CCM”) and click “OK”.

Next, go back to the Kodi home screen and click on “Add-ons”, followed by the “Add-on browser” icon (which looks like an open box). From there, click on “Install from zip file” and select the repository you just added. Install the repository and then go back to the Add-on browser menu and click on “Install from repository”. Find the Pulse CCM Repository and install it.

Finally, go back to the Kodi home screen and click on “Add-ons”. You should now see Pulse CCM listed - simply click on it to launch the build.
