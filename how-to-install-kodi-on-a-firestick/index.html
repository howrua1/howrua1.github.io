How to Install Kodi on a Firestick?

The process of installing Kodi on a firestick is fairly simple. Firstly, you need to make sure that you have enabled the option to download and install third-party apps. This can be done by going to Settings > My Fire TV > Developer Options > Apps from Unknown sources > Turn On.

Once you have completed this, you can begin the process of downloading and installing Kodi. There are several methods of doing this, but the easiest one is using the Downloader app. You can download this by going to Search > Type ‘Downloader’ > Download and Install.

After installing the Downloader app, launch it and type in the URL for downloading Kodi. Click Download and wait for the installation to finish. Once it is done, click Install. Kodi will now be installed on your firestick!
