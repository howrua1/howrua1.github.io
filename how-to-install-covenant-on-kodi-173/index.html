The first step in installing Covenant on Kodi 17.3 is to enable unknown sources. To do this, click on the gear icon on the Kodi home screen and navigate to System Settings followed by Add-ons. You will then see an option called Unknown sources. Turn it on to enable the installation of third-party addons.

After that, you need to install the Kodil Repository. To do this, go to the System Settings again and then click on File Manager. Click on Add source, and enter the URL “http://www.lvtvv.com/repo/". Name it Kodil and click on OK.

Installing Covenant on Kodil Repository

Now let’s move onto installing Covenant on the Kodil Repository. Here are the steps:

1. Go back to the home screen of Kodi and click on Add-ons.

2. Click on the Package Installer Icon located in the top left corner of the screen.

3. Select Install from Zip File and select Kodil Repository.

4. Select Kodil.zip and wait for the notification message to appear saying that the repo is installed.

5. Click on Install from Repository and select Kodil Repository.

6. Select Video add-ons and then select Covenant.

7. Click on Install, wait for the notification message, and then access the add-on from the Videos section.
